# Templates

Each file in this folder is a standalone Reveal.js template.

## Naming convention

Use kebab-case and include style direction:

- vercel-dark.html
- clean-light.html
- newsroom-contrast.html

## Lifecycle

- New ideas start in templates/drafts/.
- Promoted templates move to templates/ root.

## Notes

- Keep template files presentation-ready with placeholders.
- Prefer CDN-based Reveal.js loading unless the repo later switches to local dependencies.
